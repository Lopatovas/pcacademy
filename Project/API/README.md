# Football App API
The main goal of the API is to allow interactions with the database.
## The API has:
JWT authentication.
Password hashing.
Fully functional tests.
## Getting Started
Run the following command:
`npm i`

Prerequisites
MongoDB, Robo3T
## Built With
Node.js
NPM
express
MongoDB
